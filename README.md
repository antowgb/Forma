# Forma - Minimalist Smart Workout Generator

**Simple. Intelligent. Effective.**

Forma automatically generates your optimal workout of the day, based on recovery, favorites, equipment, and duration.

---

## Why Forma?

- Know which muscles to train today
- Avoid overtraining
- Get a ready-to-go daily workout
- Prioritize favorite exercises
- Train anywhere: gym or calisthenics

---

## Core Features

- Daily auto-generated workouts
- Smart algorithm (recovery / favorites / modality / equipment)
- Favorites system
- Local recovery tracker (per muscle readiness + decay)
- Duration presets (30 / 60 / 90 / 120 min)
- Custom workouts (build, reorder, save locally, mark complete)
- Minimalist UI with smooth screen transitions
- No account needed

---

## How It Works

1. Pick a duration
2. Forma generates your session
3. Train
4. Mark Completed -> recovery updates

---

## Smart Algorithm

Considers:

- Recovery state
- Favorites
- Training style
- Exercise duration
- Max 3 muscle groups
- Small randomness for variety

---

## Screens

### Home

Auto workout grouped by muscle, completion button, rest timer, quick reload.

### Workouts

Browse exercises, filter by modality and muscle cluster, toggle favorites.

### Recovery

Real rest time per muscle, progress bars, readiness status, reset option.

### Custom

List of saved custom sessions, reorder, view details, mark complete.

### Create

Create a custom workout (title, notes, focus, intensity, duration, exercises), saved locally.
